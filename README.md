Task 4: Dashboard Design – Sales Data Analysis with Power BI

Objective:
To design an interactive dashboard for business stakeholders using Power BI that provides insights into sales performance, customer behavior, and product trends.

Tools Used:
Power BI Desktop
Dataset:sales_data_sample.csv (from Kaggle)

Key Features of the Dashboard:

Page 1 – Executive Summary
- Total Sales
- Total Orders
- Total Customers
- Average Sales per Customer
- KPIs displayed using Cards
- Slicers for interactivity (Product Line, Country, Year)

Page 2 – Sales Trends
- Time-series line chart of monthly sales
- Sales breakdown by product line and country
- Dynamic slicers for year and customer name

Page 3 – Customer & Regional Insights
- Top 10 Customers by Sales (bar chart)
- Sales by Country (map/column chart)
- Order frequency per customer (table)
- KPI cards for Total Customers, Top Region, and Avg Sales

KPIs Chosen:
- Total Sales
- Total Orders
- Sales Growth
- Average Sales per Customer
- Top Product Line
- Regional Sales Distribution

Deliverables:
- Power BI .pbix file
- Cleaned dataset (sales_data_sample.csv)
- Screenshots of dashboard pages (for presentation)
- PowerPoint summary (if applicable)

Outcome:
An interactive and user-friendly Power BI dashboard that enables quick insights for decision-making by analyzing customer, sales, and regional performance trends.
